Here I use formules of solving quadratic equation\+:

General form is $ ax^2 + bx + c = 0 $

This function return result if $ a \neq 0 $

First i calculate discriminant\+: $ D = b^2 - 4ac$ Then I compare it with 0\+:
\begin{DoxyItemize}
\item If $ D > 0 $ Its two roots, then I use formula\+: $ x_{1,2} = \frac{-b \pm \sqrt{D}}{2a} $ and its values assigned to struct of roots.
\item If $ D = 0 $ Its one root, then I use formula\+: $ x = \frac{-b}{2a} $ and it value assigned to struct of roots, then second root is NAN.
\item If $ D < 0 $ Its no solution, then 1 and 2 roots are NAN. 
\end{DoxyItemize}
