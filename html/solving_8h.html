Additional function for solving special case of linal equation. a = 0.

Parameters
[in]argsDouble array of coefficients
[out]solsStructure for writing solutions(structure Roots)

This function called if \( a = 0 \)

If \( b \neq 0 \), then form of equation is \( bx + c = 0 \) and solution is \( x = \frac{-c}{b} \)

If \( b = 0 \), then form of equation is \( c = 0 \) and case:

  • \( c = 0 \), then \( 0 = 0 \) and its infinity of roots.
  • \( c \neq 0 \), then no roots.

double *  args

Main function of solving quadratic equation.

Parameters
[in]argsDouble array of coefficients
[out]solsStructure for writing solutions(structure Roots)

Here I use formules of solving quadratic equation:

General form is \( ax^2 + bx + c = 0 \)

This function return result if \( a \neq 0 \)

First i calculate discriminant: \( D = b^2 - 4ac\) Then I compare it with 0:

  • If \( D > 0 \) Its two roots, then I use formula: \( x_{1,2} = \frac{-b \pm \sqrt{D}}{2a} \) and its values assigned to struct of roots.
  • If \( D = 0 \) Its one root, then I use formula: \( x = \frac{-b}{2a} \) and it value assigned to struct of roots, then second root is NAN.
  • If \( D < 0 \) Its no solution, then 1 and 2 roots are NAN.
